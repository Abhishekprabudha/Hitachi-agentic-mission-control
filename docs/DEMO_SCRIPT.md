# 7-minute demo script

## 0:00 - Open with the POV
"This is not a dashboard. It is a governed agentic control plane for the GCC. Agents sense, reason, recommend, execute with approval, and audit every action."

## 1:00 - Mission Control
Show GCC health, cloud waste, K8s health, DR readiness and governance KPIs.

## 2:00 - Run FinOps anomaly
Run **Cloud spend anomaly**. Show recommendation, savings, approval and audit trail.

## 3:00 - Run K8s incident
Run **Kubernetes outage risk**. Show signal → reason → action → learning flow.

## 4:00 - Modernization factory
Open Scenarios / Modernization backlog. Explain how the system creates ranked Japan/APAC approval packets.

## 5:00 - Industrial AI platform
Show IoT pipelines. Run IoT drift scenario and explain data quality + model readiness.

## 6:00 - Governance and evidence
Open Governance. Show policies, audit trail, DR readiness and vendor SLA tables.

## 7:00 - Export evidence
Export JSON store and HTML evidence report. Close with: "Every action is explainable, approval-aware and auditable — suitable for India execution and Japan governance."
